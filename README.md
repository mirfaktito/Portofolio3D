Halaman portofolio modern yang dibangun dengan fokus pada pengalaman visual interaktif menggunakan **Three.js**. Proyek ini menampilkan integrasi antara animasi 3D, desain responsif, dan teknologi web modern.

## 🚀 Demo
[Tonton Live Demo di Sini](https://mirfaktito.github.io/Portofolio3D/) *(Pastikan kamu sudah mengaktifkan GitHub Pages)*

## 🛠️ Tech Stack
* **Core:** [Three.js](https://threejs.org/) (untuk background partikel interaktif & objek 3D)
* **Fullstack:** Laravel & Vue.js
* **Styling:** Tailwind CSS & Syne Fonts
* **Animations:** GSAP / CSS Transitions & Intersection Observer (Scroll Reveal)

## ✨ Fitur Utama
* **3D Particle Background:** Partikel interaktif yang merespons pergerakan mouse.
* **Custom Immersive Cursor:** Kursor khusus dengan efek blending mode untuk navigasi yang unik.
* **Responsive Design:** Tampilan yang sudah dioptimasi untuk perangkat mobile, tablet, dan desktop.
* **Project Showcase Cards:** Kartu proyek dengan scene 3D mini yang berbeda di setiap kartu.
* **Scroll Reveal:** Animasi halus saat elemen muncul ketika di-scroll.
